自适应天气感知图像处理 IP 设计

全国大学生集成电路创新创业大赛华南赛区二等奖

针对智能监控、无人机感知等系统在复杂多变天气环境下的图像质量退化与算法适应性不足问题，我们团队基于国产 Robei EDA 平台研发了“环境感知 - 算法决策 - 硬件加速”三级闭环的自适应天气感知图像处理 IP 核。

项目简介

  • 针对智能监控、无人机感知等系统在复杂多变的天气环境下，如雾霾、强光、夜间低照度等场景中图像质量退化、算法适应性差的行业痛点，我们团队自主研发了一款“自适应天气感知图像处理 IP 核”。
  • 项目基于国产 Robei EDA 平台，构建“环境感知 - 算法决策 - 硬件加速”三级闭环架构，实现从天气识别到图像增强、再到硬件部署输出的完整链路。

核心技术架构

  • 多模态环境感知：结合轻量化 WeatherNet 与温湿度、光照、降水等多传感器信息，实现晴、雨、雾等天气状态的实时识别。
  • 动态算法决策：基于 AXI 总线协议自研动态调度引擎（LDS），支持根据实时感知结果自动切换 18 种 ISP 处理算法链，使系统在极端天气下检测率较传统方案提升 60%。
  • 全硬件加速实现：核心算法采用纯 Verilog 底层 RTL 设计，在双 FPGA（RX50T 与 RX7020）上实现高性能流水线部署，支持 720P@30fps 实时高清图像输出。

我的贡献

  • 软硬件交互系统开发（PyQt5）：基于 PyQt5 自主设计并研发跨平台上位机控制软件，实现硬件端与用户端高效交互；构建闭环交互流程，支持实时配置底层算法链、动态调节参数阈值与直观预览增强效果，提升调试效率和用户体验。
  • 复杂算法部署与 FPGA 软核调度：主导基于 MobileNet 重构的轻量化天气识别模型在 FPGA 端落地，通过定点化量化处理与算子优化，在有限资源下实现高精度环境感知；深度参与 18 种高性能 ISP 图像处理算法的硬件逻辑构建，并完成决策层对硬件加速模块的动态调度设计。
  • 全流程硬件原型构建与工程化验证：负责项目从零到一的硬件闭环，包括原理图设计、PCB 协同布线、硬件打板验证及 3D 建模封装；主导系统在多种真实复杂气象场景下的功能性指标测试，针对资源占用与处理时延进行优化，确保 IP 核具备工程落地所需的高可靠性与实时性。

项目成果

  • 成功构建了“采集 - 处理 - 传输 - 识别”闭环的高带宽视频系统，为复杂天气与边缘感知场景下的智能巡检提供了高可靠、可复用的工程范例。
