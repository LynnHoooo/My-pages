Project Detail

基于 LoRa 通信的智能柚园巡检机器人

大学生创新训练项目，优秀等级提前结项

该项目面向果园巡检场景，结合 STM32 控制、LoRa 远距离通信和病虫害识别能力，探索农业智能装备从底层控制到智能分析的完整实现路径。

查看项目汇报

项目内容

  • 面向果园巡检任务设计机器人系统，覆盖移动控制、无线通信、识别算法与网页展示。
  • 通过 LoRa 实现远距离、抗干扰的数据传输，保证巡检过程中的稳定通信。
  • 项目后期引入 YOLOv8 病虫害识别模型，提升巡检过程中的智能化能力。

我负责的工作

  • 作为团队负责人之一，推进项目整体进度、任务拆分与阶段性交付。
  • 完成基于 STM32 的小车运动控制系统开发与联调，支持基础运动与作业逻辑实现。
  • 独立负责 LoRa 通信模块的集成与调试，解决远距离传输中的稳定性问题。
  • 参与 YOLOv8 病虫害识别接入与交互式网页设计，推动项目形成完整展示闭环。

项目亮点

  • 项目兼具机械、控制、通信与算法特征，能够体现跨方向协同开发能力。
  • 从控制系统到识别展示均有实际落地，工程链路较完整。
  • 最终以前期结项和优秀等级完成，相关成果沉淀为软著与专利材料。
