Project Detail

SoC 异构架构图像传输系统

全国大学生嵌入式芯片与系统设计竞赛国家一等奖

针对地震、核辐射、深海等极端环境下感知延迟高、通信不稳的痛点，开发了一套具备高实时性与高可靠性的全栈式探测解决方案。

项目背景

  • 针对地震、核辐射、深海等极端环境下机器人系统普遍存在的感知延迟高、通信不稳定等问题，设计一套面向复杂工况的全栈式探测方案。
  • 项目目标是在极端环境下实现高实时性的视频感知、高可靠性的远程传输以及具备初步环境理解能力的边缘智能处理。

核心技术架构

  • 计算平台：基于安路 PH1A FPGA 与 TD6.2 EDA 工具，构建“FPGA + SoC”异构协同架构。利用 FPGA 的并行处理能力实现硬件级加速，显著降低系统延迟并提升能效比。
  • 视觉处理（ISP Pipeline）：在 FPGA 内部自研全流程 ISP 算法链，集成去噪、白平衡、伽马校正、色彩增强等模块，攻克了极端环境下低照度、高动态的成像难题。
  • 通信与控制：通过千兆以太网实现毫秒级高清视频传输，确保控制指令与图像实时同步；同时集成 Wi-Fi 无线图传与多链路冗余机制，增强复杂工况下的通信健壮性。
  • 边缘智能（AI）：在 FPGA / SoC 单元部署轻量化 LeNet / MobileNet 神经网络，实现目标实时识别与分类，赋予机器人初步的环境语义理解能力。

项目成果

  • 成功构建了“采集 - 处理 - 传输 - 识别”闭环的高带宽视频系统。
  • 为极端环境下的智能巡检提供了高可靠、可复用的工程范例。
