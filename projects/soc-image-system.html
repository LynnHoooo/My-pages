Project Detail

SoC 异构架构图像传输系统

全国大学生嵌入式芯片与系统设计竞赛国家一等奖

该项目围绕图像采集、硬件级 ISP 处理、网络传输与边缘识别构建完整系统，核心目标是在异构架构下实现更高实时性、更强稳定性与更好的 系统联动能力，适用于复杂环境下的图像处理与远程交互场景。

项目内容

  • 基于 FPGA 与 SoC 的异构协同思路，完成图像采集、处理、传输和识别的一体化系统设计。
  • 系统覆盖 ISP 图像处理、网络传输、上位机交互与边缘侧轻量模型部署，形成完整闭环。
  • 项目强调实时性与系统可靠性，适合在复杂环境中承担图像感知与远程控制任务。

我参与的重点工作

  • 参与系统整体架构设计，围绕 FPGA 与 SoC 的分工方式推进功能落地与模块联调。
  • 参与图像采集与 ISP 处理链路的实现与优化，关注处理流程稳定性和显示效果。
  • 配合完成网络传输、边缘识别与上位机控制之间的对接，推动完整系统跑通。

项目亮点

  • 突出异构协同能力，把底层硬件处理与上层系统功能结合到同一平台中。
  • 强调系统级联调，不只是单点算法或单模块实现，而是完整链路打通。
  • 最终成果获得国家一等奖，体现了项目在方案设计、工程实现和展示效果上的综合竞争力。
